## Requisitos adicionais da entrega

Além da construção dos visuais principais, o relatório também contempla pontos importantes solicitados na atividade, garantindo melhor navegação, clareza e experiência para o usuário final.

### Pontos de atenção implementados

| Item | Descrição |
|---|---|
| Estrutura definida | O relatório foi organizado com uma estrutura visual clara, facilitando a leitura e a interpretação dos dados. |
| Botões de navegação | Foram utilizados botões de navegação para permitir a troca entre páginas do relatório, proporcionando melhor navegabilidade. |
| Segmentadores utilizados | Foram adicionados segmentadores para permitir filtros dinâmicos nas análises. |
| Botões com imagem associada | Alguns botões foram configurados com imagens associadas, tornando a navegação mais visual e intuitiva. |
| Indicadores e botões interativos | Foram utilizados indicadores e botões para permitir a seleção de diferentes visuais sobre o mesmo assunto. |

---

## Navegação e interatividade

O relatório foi desenvolvido com foco em usabilidade, permitindo que o usuário navegue entre as páginas e alterne diferentes visões analíticas de forma simples.
