## Visão geral do projeto

Este projeto foi desenvolvido como parte de um desafio prático com foco na criação de um relatório analítico em **Power BI**, utilizando dados de vendas, unidades vendidas e lucro por país e segmento.

O objetivo foi construir uma página de relatório clara, visual e objetiva, permitindo a análise de desempenho comercial por região e por tipo de cliente/segmento.

A terceira página do relatório foi composta por visuais voltados para análise geográfica e segmentada, considerando vendas, unidades, lucro e participação por segmento.
